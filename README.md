# CattaBCU

### Catta Bike Control Unit

### Hardware:
- MCU: STM32F030K6T6
- Display controller: SH1106

### Screens:
- Main:
    - actual speed
    - trip time
    - trip distance
    - battery level
    - temperature
    - lights on icon
    - braking icon

- Statistics:
    - total distance
    - total time
    - max speed

- Front light settings:
    - normal pwm value
    - boost pwm value

- Rear light settings:
    - normal pwm value
    - braking pwm value
